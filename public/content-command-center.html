Spotify for Podcasters — Setup

1. Go to podcasters.spotify.com and sign up with 4UPact Google account
2. Create show: "The Growth Engine Show" — add cover art (1400×1400px min)
3. Set language to English (create separate shows for ES/PT or use multilingual episodes)
4. Upload first episode MP3 + title + description + show notes
5. Copy the RSS feed URL — this is your universal distribution key
6. Connect RSS to Make.com: New File in Google Drive → HTTP POST to RSS endpoint
FREERSS = Universal KeyAuto-index 2–4h
🍎

Apple Podcasts Connect — Setup

1. Go to podcastsconnect.apple.com — requires Apple ID
2. Click "+" → Paste the same RSS feed URL from Spotify for Podcasters
3. Apple validates and indexes the feed (takes 24–48h first time)
4. Once approved, every new episode auto-appears on Apple Podcasts
5. No manual uploads needed — RSS handles everything
6. Add show to Apple categories: Business → Marketing / Entrepreneurship
FREESame RSS Feed24–48h first index
📺

YouTube Podcasts — Setup

1. YouTube Studio → Settings → Channel → Enable Podcasts feature
2. Option A: Submit RSS feed (YouTube auto-imports episodes)
3. Option B: Make.com automation → YouTube Data API v3 → auto-upload MP3 + thumbnail
4. Add video description template with timestamps + links + CTA (sales@4upact.com)
5. Create 3 playlists: EN Episodes, ES Episodes, PT Episodes
FREERSS or APISEO via YouTube search

Make.com Orchestration — Setup
